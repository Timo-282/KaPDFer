PDF → DOCX (rein client-seitig, für GitHub Pages)

Bereit
Hinweis:
  • Diese Seite läuft komplett im Browser — deine PDF-Datei verlässt deinen Rechner nicht.
  • Für gescannte PDFs (Bilder) ist OCR nötig (z. B. Tesseract.js); diese Datei extrahiert nur echten Text aus PDFs.
  • Layout, Bilder und komplexe Tabellendarstellungen werden nicht 1:1 übernommen — nur Rohtext.
Protokoll / Ausgaben 
—
Datei auf GitHub hochladen (index.html) → Pages aktivieren (Branch: main oder gh-pages).
Wenn du OCR brauchst, sag Bescheid — ich kann dir eine Version mit Tesseract.js vorbereiten.
